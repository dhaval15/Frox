## Flux

It is setup made with awesome window manager.

#### Dependencies

* Awesome WM
* Compton
* Rofi
* Alsa
* Xbacklight
* Gnome Terminal

#### Features

- [x] Side Bar
- [x] Tag Bar
- [x] Start Screen
- [x] Exit Screen
- [ ] Task Bar
- [ ] Minimal Window Decorations

#### Bugs 

- Exit Screen Icons Mismatch
- Start Screen Colour Contrast
- Window Rules Causing Dialog Problems
